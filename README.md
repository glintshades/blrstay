# Blrstay — Premium Men's PG in Yelahanka, Bangalore

  Professional website for Blrstay PG accommodation in Yelahanka, Bangalore.

  ## Tech Stack
  - React + TypeScript + Tailwind CSS + Shadcn UI
  - Express.js + Node.js backend
  - PostgreSQL with Drizzle ORM

  ## Features
  - Multi-page: Home, Rooms, Amenities, Location, Contact
  - Online booking enquiry system
  - Admin panel with analytics, room/tenant/facility management
  - SEO-optimized for local search
